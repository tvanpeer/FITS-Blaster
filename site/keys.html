Keyboard Shortcuts

All default key bindings — every shortcut is reconfigurable in Settings → Keyboard.

Navigation

KeyAction
↑ Previous image — move the cursor to the previous frame in the sidebar.
↓ Next image — move the cursor to the next frame.
Home First image — jump to the first frame in the current view.
End Last image — jump to the last frame.
⇧+↑ Extend selection up — add the previous frame to the orange range selection and move the cursor there.
⇧+↓ Extend selection down — add the next frame to the orange range selection.

Rejection

KeyAction
X Reject — moves the current frame (or the entire orange range) to REJECTED/. In toggle mode, also acts as undo if the frame is already rejected.
U Undo rejection — moves the current frame back from REJECTED/. Only available when Single key reject/undo is off in Settings.

Enable Single key reject/undo (toggle) in Settings → Keyboard to use one key for both reject and undo — useful when the spacebar or a gamepad button is mapped as the reject key.

Flagging

KeyAction
F (De)flag — toggles the flag state of the current frame. If an orange range is active, the entire range is flagged or unflagged at once. Flagged frames appear in the Flagged view.
D Deflag all — removes all frames from the Flagged view in one keystroke. Does not reject anything.
P Play / Pause — starts or stops auto-advancing through images at the configured speed. Any manual navigation key stops playback.
V Flip 180° — rotates the current image (or selection) 180 degrees for display. Does not modify the FITS file. Press again to unflip.

⌘+click in the sidebar is an alternative way to flag and unflag: in the All view it adds a frame (or the orange range) to the Flagged view; in the Flagged view it removes it.

Range Selection

The orange range is used for batch operations: reject, flag, or unflag a group of frames at once. It is independent of the cursor (blue highlight).

KeyAction
⌘+A Select all visible — orange-selects every frame in the current view.
⌘+D Deselect all — clears the orange range selection.
⌘+I Invert selection — selects all visible frames that are not currently in the orange range, and deselects those that are.
⌘+R Select all rejected — orange-selects every rejected frame visible in the current view.

All four shortcuts can be changed to ⌘⇧+key in Settings → Keyboard. They always operate on the orange range, never on the Flagged view membership.

View & Display

KeyAction
G Toggle Simple / Geek mode — Simple mode hides metrics and the session chart for a faster visual pass; Geek mode shows all numeric data. The toolbar button shows the current mode name.
C Toggle colour images — switches between greyscale and colour (debayered) rendering for OSC cameras. Mono cameras are always shown in greyscale.

File Operations

KeyAction
⌘+O Open Folder… — opens a folder picker. All FITS files in the selected folder are loaded immediately.
R Remove from list — removes the current frame from the session list without touching the file on disk.

Mouse & Click Shortcuts

ActionEffect
Click (sidebar) Move the cursor to that frame; clear the orange range.
⇧+click Select an orange range from the last-clicked frame to the clicked frame.
⌘+click In All view: add the frame (or orange range) to the Flagged view.
In Flagged view: remove the frame (or orange range) from the Flagged view.
Click (chart dot) Move the cursor to that frame in the viewer.
Drag (chart) Select a run of frames and add them to the Flagged view.
